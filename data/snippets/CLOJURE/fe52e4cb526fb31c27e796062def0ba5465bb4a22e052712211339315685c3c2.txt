(defn add-numbers 
     "This function takes in two numbers and returns their sum" 
     [number1 number2] 
     (+ number1 number2))
;;the `defn` keyword is used to define a function in Clojure
;;the function is named `add-numbers` 
;;the `defn` keyword is followed by the name of the function and 
;;a vector containing the parameters of the function 
;;in this case, the parameters are named `number1` and `number2` 
;;the `+` operator is used to add the two numbers 
;;the resultant sum is returned as the output of the function
