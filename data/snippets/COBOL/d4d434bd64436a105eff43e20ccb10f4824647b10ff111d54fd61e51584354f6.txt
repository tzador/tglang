IDENTIFICATION DIVISION.
      PROGRAM-ID. EXAMPLE.
      DATA DIVISION.
      WORKING-STORAGE SECTION.
      01 INPUT-VALUE PIC 9(10)V99.
      01 CALCULATED-VALUE PIC S9(5)V99.
      01 ERROR-FLAG PIC X VALUE "N".

      PROCEDURE DIVISION.
          ACCEPT INPUT-VALUE.
          IF (INPUT-VALUE < 0) OR (INPUT-VALUE > 1000)
              MOVE "Y" TO ERROR-FLAG
          ELSE
              PERFORM CALCULATE-VALUE.
          IF ERROR-FLAG = "Y"
              DISPLAY "ERROR: Input value must be between 0 and 1000"
          ELSE
              DISPLAY "Calculated value: " CALCULATED-VALUE.
          STOP RUN.
          
      CALCULATE-VALUE.
          MULTIPLY INPUT-VALUE BY 7.35 GIVING CALCULATED-VALUE.
