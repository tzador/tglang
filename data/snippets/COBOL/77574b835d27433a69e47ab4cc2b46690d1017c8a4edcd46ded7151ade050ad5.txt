IDENTIFICATION DIVISION.
PROGRAM-ID. GENERATED-CODE-SNIPPET.

ENVIRONMENT DIVISION.
DATA DIVISION.
WORKING-STORAGE SECTION.
  01 INPUT-DATA.
    05 NUMBER PIC 9(5) VALUE 12345.
    05 FIRST-NUMBER PIC 9(3).
    05 SECOND-NUMBER PIC 9(3).
  01 OUTPUT-DATA.
    05 RESULT PIC 9(5).

PROCEDURE DIVISION.
MAIN-PROCEDURE.
  PERFORM CALCULATE-RESULT
  DISPLAY "The result is " RESULT
  STOP RUN.

CALCULATE-RESULT.
  COMPUTE RESULT = FIRST-NUMBER + SECOND-NUMBER
  IF RESULT NOT = NUMBER
    SET RESULT TO NUMBER
  END-IF.
