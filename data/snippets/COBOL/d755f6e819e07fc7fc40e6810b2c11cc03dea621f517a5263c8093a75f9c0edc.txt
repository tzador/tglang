IDENTIFICATION DIVISION.
       PROGRAM-ID. CALCULATE-PAYROLL.

       ENVIRONMENT DIVISION.
       INPUT-OUTPUT SECTION.
       FILE-CONTROL.
           SELECT EMPLOYEE-FILE ASSIGN TO "EMPLOYEES.DAT".

       DATA DIVISION.
       FILE SECTION.
       FD EMPLOYEE-FILE
           RECORD CONTAINS 30 CHARACTERS
           LABEL RECORD IS STANDARD
           DATA RECORD IS EMPLOYEE-RECORD.
       01 EMPLOYEE-RECORD.
           05 EMPLOYEE-NAME     PIC X(20).
           05 EMPLOYEE-AGE      PIC 99.
           05 EMPLOYEE-SALARY   PIC 9(7)V99.

       WORKING-STORAGE SECTION.
       01 TOTAL-SALARY       PIC 9(9)V99 VALUE 0.
       01 TOTAL-EMPLOYEES    PIC 99 VALUE 0.
       01 TAX-RATE           PIC 9(3)V9(2) VALUE 0.15.

       PROCEDURE DIVISION.
       100-MAIN-ROUTINE.
           OPEN INPUT EMPLOYEE-FILE.
           READ EMPLOYEE-FILE
               AT END GO TO 200-PROGRAM-END.
       150-PROCESS-EMPLOYEE.
           ADD EMPLOYEE-SALARY TO TOTAL-SALARY.
           ADD 1 TO TOTAL-EMPLOYEES.
           MULTIPLY EMPLOYEE-SALARY BY TAX-RATE GIVING EMPLOYEE-TAX.
           DISPLAY "Employee Name: " EMPLOYEE-NAME.
           DISPLAY "Employee Age: " EMPLOYEE-AGE.
           DISPLAY "Employee Salary: " EMPLOYEE-SALARY.
           DISPLAY "Employee Tax: " EMPLOYEE-TAX.
       200-PROGRAM-END.
           CLOSE EMPLOYEE-FILE.
           DISPLAY "Total Employees: " TOTAL-EMPLOYEES.
           DISPLAY "Total Payroll: " TOTAL-SALARY.
           STOP RUN.
