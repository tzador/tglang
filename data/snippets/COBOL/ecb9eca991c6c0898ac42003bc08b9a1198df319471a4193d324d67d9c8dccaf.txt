IDENTIFICATION DIVISION.
PROGRAM-ID. CALCULATE-AREA.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 LENGTH PIC 9(3) VALUE 0.
01 WIDTH PIC 9(3) VALUE 0.
01 AREA PIC 9(5)V99 VALUE 0.

PROCEDURE DIVISION.
CALCULATE-AREA.
    DISPLAY "Enter length:".
    ACCEPT LENGTH.
    DISPLAY "Enter width:".
    ACCEPT WIDTH.

    IF LENGTH = 0 OR WIDTH = 0
        DISPLAY "Invalid input.".
        STOP RUN.
    END-IF.

    MULTIPLY LENGTH BY WIDTH GIVING AREA.
    DISPLAY "Area is: " AREA.
    STOP RUN.

END PROGRAM CALCULATE-AREA.
