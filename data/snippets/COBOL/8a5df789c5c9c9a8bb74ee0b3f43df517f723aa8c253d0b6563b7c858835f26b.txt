IDENTIFICATION DIVISION.
PROGRAM-ID. PROGRAM1.

* This is a comment line

DATA DIVISION.
WORKING-STORAGE SECTION.
01 COUNTER PIC 9(3) VALUE 0.
01 INPUT-STRING PIC X(20).
01 OUTPUT-STRING PIC X(20).
01 RESULT PIC 9(3) VALUE 0.

PROCEDURE DIVISION.
BEGIN-PROGRAM.

* Loop through 10 times
PERFORM 10 TIMES
    DISPLAY "Enter a string: ".
    ACCEPT INPUT-STRING.

    * Check if string contains only numbers
    INSPECT INPUT-STRING TALLYING COUNTER FOR CHARACTERS BEFORE '.' IS NUMERIC
    IF COUNTER = 20 THEN
        DISPLAY "String is valid".
        MOVE 1 TO RESULT
    ELSE
        DISPLAY "String is invalid".
        MOVE 0 TO RESULT
    END-IF

    * Check if string contains "COBOL"
    INSPECT INPUT-STRING TALLYING COUNTER FOR ALL "COBOL"
    IF COUNTER > 0 THEN
        DISPLAY "String contains COBOL".
        ADD 1 TO RESULT
    END-IF
    MOVE INPUT-STRING TO OUTPUT-STRING
END-PERFORM.

* Display final result
DISPLAY "Final result is " RESULT.

STOP RUN.
