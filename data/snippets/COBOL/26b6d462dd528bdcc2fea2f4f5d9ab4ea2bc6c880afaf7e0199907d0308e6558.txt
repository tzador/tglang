IDENTIFICATION DIVISION.
PROGRAM-ID. EXAMPLE.
ENVIRONMENT DIVISION.
INPUT-OUTPUT SECTION.
FILE-CONTROL.
SELECT IN-FILE ASSIGN TO '/Users/example/inputfile.txt'.
SELECT OUT-FILE ASSIGN TO '/Users/example/outputfile.txt'.

DATA DIVISION.
FILE SECTION.
FD IN-FILE.
01 INPUT-RECORD PIC X(80).

FD OUT-FILE.
01 OUTPUT-RECORD PIC X(80).

WORKING-STORAGE SECTION.
01 RECORD-COUNTER PIC 99 VALUE 0.

PROCEDURE DIVISION.
OPEN INPUT IN-FILE.
OPEN OUTPUT OUT-FILE.

WHILE RECORD-COUNTER < 100
    READ IN-FILE INTO INPUT-RECORD
    AT END
        GO TO END-OF-FILE
    NOT AT END
        PERFORM PROCESS-RECORD
    END-READ
END-WHILE.

CLOSE IN-FILE.
CLOSE OUT-FILE.

STOP RUN.

PROCESS-RECORD.
MOVE INPUT-RECORD TO OUTPUT-RECORD.
ADD 1 TO RECORD-COUNTER.
WRITE OUTPUT-RECORD.
