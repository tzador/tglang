IDENTIFICATION DIVISION.
 PROGRAM-ID. CALCULATE-PAYROLL.

 * Program to calculate payroll for employees

 DATA DIVISION.
 WORKING-STORAGE SECTION.

 01 EMPLOYEE-TABLE.
    05 ID-NUMBER        PIC X(6) VALUE '123456'.
    05 NAME             PIC X(20) VALUE 'John Smith'.
    05 HOURS-WORKED     PIC 9(6)V9(2) VALUE 40.00.
    05 HOURLY-RATE      PIC 9(5)V9(2) VALUE 10.00.
    05 GROSS-PAY        PIC 9(7)V9(2).

 PROCEDURE DIVISION.
    PERFORM CALCULATE-PAYROLL VARYING I
         FROM 1 BY 1 UNTIL I > 5.

 CALCULATE-PAYROLL.
    MULTIPLY HOURS-WORKED BY HOURLY-RATE
        GIVING GROSS-PAY.
    DISPLAY 'Employee ID: ' ID-NUMBER.
    DISPLAY 'Employee Name: ' NAME.
    DISPLAY 'Hours Worked: ' HOURS-WORKED.
    DISPLAY 'Hourly Rate: ' HOURLY-RATE.
    DISPLAY 'Gross Pay: ' GROSS-PAY.
    ADD 1 TO I.

  END-PERFORM.

  STOP RUN.
