* This program calculates the salary of an employee based on their job title and years of service
IDENTIFICATION DIVISION.
PROGRAM-ID. SALARY-CALCULATION.

* Declare input variables
DATA DIVISION.
WORKING-STORAGE SECTION.
01 JOB-TITLE PIC X(10).
01 YEARS-SERVICE PIC 9(3).
01 SALARY PIC 9(6).

* Assign job titles and corresponding salary ranges
01 MANAGER-LOWER-LIMIT PIC 9(6) VALUE 50000.
01 MANAGER-UPPER-LIMIT PIC 9(6) VALUE 80000.
01 DIRECTOR-LOWER-LIMIT PIC 9(6) VALUE 80000.
01 DIRECTOR-UPPER-LIMIT PIC 9(6) VALUE 120000.
01 EXECUTIVE-LOWER-LIMIT PIC 9(6) VALUE 120000.
01 EXECUTIVE-UPPER-LIMIT PIC 9(6) VALUE 200000.

* Read input from user
PROCEDURE DIVISION.
DISPLAY "Enter job title: ".
ACCEPT JOB-TITLE.
DISPLAY "Enter years of service: ".
ACCEPT YEARS-SERVICE.

* Calculate salary based on job title and years of service
IF JOB-TITLE = "MANAGER" THEN
    IF YEARS-SERVICE < 5 THEN
        COMPUTE SALARY = MANAGER-LOWER-LIMIT
    ELSE IF YEARS-SERVICE > 5 THEN
        COMPUTE SALARY = MANAGER-UPPER-LIMIT
    END-IF
ELSE IF JOB-TITLE = "DIRECTOR" THEN
    IF YEARS-SERVICE < 10 THEN
        COMPUTE SALARY = DIRECTOR-LOWER-LIMIT
    ELSE IF YEARS-SERVICE > 10 AND YEARS-SERVICE < 20 THEN
        COMPUTE SALARY = DIRECTOR-UPPER-LIMIT
    ELSE
        COMPUTE SALARY = DIRECTOR-UPPER-LIMIT + (YEARS-SERVICE - 20) * 5000
    END-IF
ELSE IF JOB-TITLE = "EXECUTIVE" THEN
    IF YEARS-SERVICE < 15 THEN
        COMPUTE SALARY = EXECUTIVE-LOWER-LIMIT
    ELSE IF YEARS-SERVICE > 15 AND YEARS-SERVICE < 25 THEN
        COMPUTE SALARY = EXECUTIVE-UPPER-LIMIT
    ELSE
        COMPUTE SALARY = EXECUTIVE-UPPER-LIMIT + (YEARS-SERVICE - 25) * 10000
    END-IF.

* Output the calculated salary
DISPLAY "The salary for a " JOB-TITLE " with " YEARS-SERVICE " years of service is: " SALARY.

STOP RUN.
