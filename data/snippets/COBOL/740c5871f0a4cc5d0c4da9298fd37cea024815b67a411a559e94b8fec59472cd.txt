**IDENTIFICATION DIVISION.** 
* This code snippet is a simple COBOL program that multiplies two numbers and
* prints the result.
PROGRAM-ID. MULTIPLY.
AUTHOR. YOUR NAME.
DATE-WRITTEN. TODAY'S DATE.

**ENVIRONMENT DIVISION.**
CONFIGURATION SECTION.
SOURCE-COMPUTER. IBM-PC.
OBJECT-COMPUTER. IBM-PC.

INPUT-OUTPUT SECTION.
FILE-CONTROL.

**DATA DIVISION.**
FILE SECTION.
FD INPUT-FILE.
01 INPUT-RECORD.
    02 FIRST-NUMBER PIC 9(3).
    02 SECOND-NUMBER PIC 9(3).

WORKING-STORAGE SECTION.
01 MULTIPLICATION-RESULT PIC 9(5).

**PROCEDURE DIVISION.**
MAIN-PARAGRAPH.
    OPEN INPUT INPUT-FILE
    READ INPUT-FILE
        AT END
            DISPLAY "ERROR: INPUT FILE EMPTY"
            STOP RUN
        NOT AT END
            COMPUTE MULTIPLICATION-RESULT = FIRST-NUMBER * SECOND-NUMBER
            DISPLAY "THE RESULT IS " MULTIPLICATION-RESULT
    CLOSE INPUT-FILE
    STOP RUN.
