PROCEDURE DIVISION.
PROGRAM-ID. CALCULATE-STATISTICS.
DATA DIVISION.
WORKING-STORAGE SECTION.
01 INPUT-ARRAY.
    05 ARRAY-SIZE PIC 9(3) VALUE 10.
    05 NUMERIC-ITEMS OCCURS 10 TIMES PIC 9(4).
    05 TOTAL PIC 9(4) VALUE 0.
    05 AVERAGE PIC 9(4)V9(2) VALUE 0.
    05 MAXIMUM PIC 9(4) VALUE 0.
    05 MINIMUM PIC 9(4) VALUE 0.
01 TEMP-NUMBER PIC 9(4).
01 I PIC 9(2) VALUE 1.
PROCEDURE DIVISION.
    PERFORM READ-ARRAY.
    PERFORM CALCULATE-TOTAL.
    PERFORM CALCULATE-AVERAGE.
    PERFORM CALCULATE-MAX-MIN.
    PERFORM DISPLAY-RESULTS.
STOP RUN.
READ-ARRAY.
    PERFORM VARYING I FROM 1 BY 1 UNTIL I > ARRAY-SIZE
        DISPLAY "Enter a number: "
        ACCEPT TEMP-NUMBER
        MOVE TEMP-NUMBER TO NUMERIC-ITEMS(I)
    END-PERFORM.
CALCULATE-TOTAL.
    COMPUTE TOTAL = FUNCTION SUM(NUMERIC-ITEMS).
CALCULATE-AVERAGE.
    COMPUTE AVERAGE = TOTAL / ARRAY-SIZE.
CALCULATE-MAX-MIN.
    SET MAXIMUM TO NUMERIC-ITEMS(1).
    SET MINIMUM TO NUMERIC-ITEMS(1).
    PERFORM VARYING I FROM 1 BY 1 UNTIL I > ARRAY-SIZE
        IF NUMERIC-ITEMS(I) > MAXIMUM
            SET MAXIMUM TO NUMERIC-ITEMS(I)
        END-IF
        IF NUMERIC-ITEMS(I) < MINIMUM
            SET MINIMUM TO NUMERIC-ITEMS(I)
        END-IF
    END-PERFORM.
DISPLAY-RESULTS.
    DISPLAY "Total: " TOTAL
    DISPLAY "Average: " AVERAGE
    DISPLAY "Maximum: " MAXIMUM
    DISPLAY "Minimum: " MINIMUM.
