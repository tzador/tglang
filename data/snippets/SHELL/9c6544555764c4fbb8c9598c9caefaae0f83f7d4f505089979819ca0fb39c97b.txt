# This is a shell script that calculates the area of a triangle

# Define variables for base and height of triangle
base=5
height=10

# Calculate the area using the formula (base * height)/2
area=$((base * height / 2))

# Print the result
echo "The area of the triangle is: $area"

# This is a multi-line comment to add further explanation
<<COMMENT
The above code calculates the area of a triangle by first defining the values 
for the base and height. It then uses the formula (base * height)/2 to calculate 
the area and stores the result in the variable "area". Finally, it prints the 
result using the "echo" command. 
COMMENT
