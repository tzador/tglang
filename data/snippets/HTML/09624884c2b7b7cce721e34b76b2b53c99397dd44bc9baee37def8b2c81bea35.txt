<!--
*This is a multi-line comment in HTML.
It will not be rendered in the final output.
You can use comments to add notes or explanations in your code.
It is recommended to comment your code for better readability.*
-->

<!DOCTYPE html>
<!--
*The DOCTYPE declaration should be the very first line of your HTML document.
It tells the browser which version of HTML the page is written in.
HTML5 does not require a DOCTYPE declaration, but it is still recommended for validation.*
-->
<html lang="en">
<!--
*The <html> tag represents the root of an HTML document.
It contains all other HTML elements and the <head> and <body> tags.*
-->
<head>
<!--
*The <head> element contains metadata about the document, including links to external resources and scripts.
The content inside the <head> tag is not visible on the page.*
-->
<title>My Code Snippet</title>
<!--
*The <title> tag specifies the title of the document, which is displayed in the browser's title bar or tab.*
-->
<meta charset="utf-8">
<!--
*The <meta> tag defines metadata about the HTML document, such as the character encoding.*
-->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<!--
*The <meta> tag with the name "viewport" specifies how the document should be displayed on different devices.*
-->
</head>
<body>
<!--
*The <body> element contains all visible content of the HTML document.*
-->
<header>
<h1>My Code Snippet</h1>
<!--
*The <header> tag defines a header section for the document.
<h1> is a heading element with the largest size.*
-->
<nav>
<ul>
<li><a href="#section1">Section 1</a></li>
<!--
*The <nav> tag defines navigation links for the document.
<ul> is an unordered list of items.
<li> is a list item.
<a> is a link.*
-->
<li><a href="#section2">Section 2</a></li>
<li><a href="#section3">Section 3</a></li>
</ul>
</nav>
</header>
<main>
<!--
*The <main> tag defines the main content of the document.*
-->
<section id="section1">
<h2>Section 1</h2>
<p>This is the first section of my code snippet.</p>
<!--
*The <section> tag defines a section within the document.
<h2> is a heading element with a medium size.
<p> is a paragraph element.*
-->
</section>
<section id="section2">
<h2>Section 2</h2>
<p>This is the second section of my code snippet.</p>
</section>
<section id="section3">
<h2>Section 3</h2>
<p>This is the third section of my code snippet.</p>
</section>
</main>
<footer>
<small>This is a footer</small>
<!--
*The <footer> tag defines a footer section for the document.
<small> is a small text element.*
-->
</footer>
</body>
</html>
