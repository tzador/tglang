<!DOCTYPE html> <!-- This declares the document type as HTML -->
<html> <!-- This indicates the start of the HTML document -->
<head> <!-- This allows us to add metadata for the document -->
    <title>Annotated Code Snippet</title> <!-- This gives a title for the document that appears in the browser tab -->
    <meta charset="UTF-8"> <!-- This sets the character encoding for the document to UTF-8 -->
    <meta name="description" content="Code snippet in HTML"> <!-- This gives a description for the document that appears in search engine results -->
    <style> <!-- This allows us to add CSS styling for the document -->
        body { /* This sets styles for the body element */
            font-family: Arial, sans-serif; /* This sets the font family to Arial or any sans-serif font */
            background-color: lightblue; /* This sets the background color for the body */
        }
        h1 { /* This sets styles for the h1 element */
            color: red; /* This sets the font color to red */
            font-size: 24px; /* This sets the font size to 24px */
        }
    </style>
</head>
<body> <!-- This is the start of the body section -->
    <div> <!-- This creates a div element, used for grouping elements together -->
        <h1>Code Snippet in HTML</h1> <!-- This creates a heading element with the text "Code Snippet in HTML" -->
        <p>This is an annotated code snippet in HTML. It showcases the use of different HTML elements and attributes.</p> <!-- This creates a paragraph element with the text "This is an annotated code snippet in HTML. It showcases the use of different HTML elements and attributes." -->

            <li>HTML stands for HyperText Markup Language.</li> <!-- This creates a list item with the text "HTML stands for HyperText Markup Language." -->
            <li>It is used for creating the structure and content of a webpage.</li> <!-- This creates a list item with the text "It is used for creating the structure and content of a webpage." -->
            <li>HTML uses tags and attributes to define elements and their properties.</li> <!-- This creates a list item with the text "HTML uses tags and attributes to define elements and their properties." -->
        </ul>
        <a href="https://www.w3schools.com/html/">Learn more about HTML</a> <!-- This creates a link element with the text "Learn more about HTML" and the URL "https://www.w3schools.com/html/" -->
    </div>
</body>
</html>
