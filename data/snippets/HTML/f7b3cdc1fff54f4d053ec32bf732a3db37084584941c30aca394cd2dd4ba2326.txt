<!doctype html> <!-- This is the document type declaration for HTML5 -->

<html> <!-- This is the opening tag for the HTML document -->

<head> <!-- This is the head element which contains information about the document -->
  <title>This is my code snippet</title> <!-- This is the title of the document which is displayed in the browser's tab -->
  <meta charset="UTF-8"> <!-- This is the meta element for specifying the character encoding of the document -->
  <meta name="description" content="A code snippet in HTML"> <!-- This is the meta element for specifying a description for the document -->
  <meta name="author" content="Your Name"> <!--This is the meta element for specifying the author of the document -->
</head>

<body> <!-- This is the body element which contains the visible content of the document -->

  <header> <!-- This is the header element which contains the header of the document -->
    <h1>My Code Snippet</h1> <!-- This is the heading element with level 1 which means the biggest heading -->
  </header>

  <main> <!-- This is the main element which contains the main content of the document -->

    <section> <!-- This is the section element which contains a specific section of the content -->

      <h2>Introduction</h2> <!-- This is the heading element with level 2 which means a smaller heading -->
      <p>This is a long and complex code snippet written in HTML. It is meant to showcase my coding skills.</p> <!-- This is a paragraph element that contains text -->
    </section>

    <section>
      <h2>Example Code</h2>
      <p>Below is an example of how to create a list in HTML code:</p>
      <ul> <!-- This is the unordered list element -->
        <li>Item 1</li> <!-- This is the list item element -->
        <li>Item 2</li>
        <li>Item 3</li>
      </ul>
    </section>

    <section>
      <h2>Conclusion</h2>
      <p>Creating complex code in HTML can be challenging, but with practice, it becomes easier.</p>
    </section>
  </main>

  <footer> <!-- This is the footer element which contains the footer of the document -->
    <p>Created by Your Name</p>
  </footer>

</html> <!-- This is the closing tag for the HTML document -->
