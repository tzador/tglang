' This code snippet is a simple sub-routine that calculates the average of three numbers.
Dim num1, num2, num3 As Integer ' declaring three integer variables
num1 = 5 ' assigning values to variables
num2 = 8
num3 = 12
Dim average As Double ' declaring an average variable as a Double (decimal) data type
average = (num1 + num2 + num3) / 3 ' calculating the average
Console.WriteLine("The average is: " & average) ' displaying the result
