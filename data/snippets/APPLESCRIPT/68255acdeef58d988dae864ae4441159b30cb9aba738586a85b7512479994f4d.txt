# This code is used to find the sum of numbers

set num1 to 10 # Assigns the value 10 to the variable num1
set num2 to 20 # Assigns the value 20 to the variable num2

set sum to num1 + num2 # Adds num1 and num2 and assigns the result to the variable sum

display dialog "The sum of " & num1 & " and " & num2 & " is " & sum # Displays a dialog pop-up with the sum

# Output: The sum of 10 and 20 is 30
