-- This code snippet is written in ICON programming language.
-- It is a basic program that calculates the area of a rectangle.

-- Initialize variables
num::width, height, area
width := 10  -- Assign a value of 10 to width variable
height := 5  -- Assign a value of 5 to height variable

-- Calculate the area of rectangle and store it in area variable
area := width * height

-- Print the result
write("The area of the rectangle is: ", area)
