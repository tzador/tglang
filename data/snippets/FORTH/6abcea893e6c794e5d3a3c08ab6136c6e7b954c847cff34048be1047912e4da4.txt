\ complexity

: BIG-O \ A FORTH WORD THAT TAKES TWO INPUTS AND OUTPUTS THE BIG-O NOTATION OF THE COMPLEXITY OF THE FUNCTION
\ INPUTS
\ N - A VARIABLE REPRESENTING THE SIZE OF THE INPUT
\ M - A VARIABLE REPRESENTING THE NUMBER OF TIMES THE FUNCTION IS EXECUTED

: BIG-O ( N M -- NOTATION )
  2DUP             \ DUPLICATE THE TWO INPUTS ON THE STACK
  = IF             \ CHECK IF THEY ARE EQUAL, IF SO
      1 SWAP       \ MOVE THE 1 ON TOP OF THE STACK
      SWAP         \ SWAP THE INPUTS BACK
  ELSE IF          \ IF THEY ARE NOT EQUAL, THEN
      SWAP         \ SWAP THE INPUTS
      2DUP         \ DUPLICATE THE INPUTS AGAIN
      /MOD         \ DIVIDE THE FIRST INPUT BY THE SECOND AND GET THE REMAINDER
      1+           \ ADD 1 TO THE REMAINDER, ACCOUNTING FOR THE EXTRA STEP
      SWAP         \ SWAP THE REMAINDER AND THE INPUTS
  THEN             \ END IF STATEMENT
  DUP IF           \ DUPLICATE THE INPUT AND CHECK IF IT IS NON-ZERO
      "O(N)"       \ IF IT IS, RETURN O(N) NOTATION
  ELSE             \ IF IT IS ZERO
      DROP         \ DISCARD THE INPUT
      "O(1)"       \ RETURN O(1) NOTATION
  THEN ;           \ END WORD DEFINITION
