: FIB             ( n1 -- n2)
  DUP IF          ( duplicate n1 if not zero)
    1- FIB FIB + ( recursive call for n1-1 and n1-2, then add)
  ELSE            ( if n1 is zero, return 1)
    DROP 1        ( drop n1 and return 1)
  THEN ;          ( end if statement)
