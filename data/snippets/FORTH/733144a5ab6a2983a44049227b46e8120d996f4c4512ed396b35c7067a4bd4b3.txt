: add ( adds two numbers together )
  dup ( duplicates the top value on the stack )
  + ( adds top two values on stack )
; ( end of word definition )
: square ( squares a number )
  dup * ( multiplies top value on stack by itself )
; ( end of word definition )
: quadruple ( multiplies a number by four )
  dup square square ( uses square word twice to square number twice )
; ( end of word definition )
5 3 add quadruple . ( pushes 5 and 3 onto stack, adds them, squares the result, and prints it )
