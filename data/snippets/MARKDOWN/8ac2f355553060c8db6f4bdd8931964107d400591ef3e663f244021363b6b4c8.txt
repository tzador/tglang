## My Markdown Snippet

> Markdown is a lightweight markup language commonly used for creating formatted text on the web. It is designed to be easy to read and write, while still allowing for advanced customization and styling.

1. Create a new markdown document.
2. Start with a level 1 heading, using one or more `#` symbols before the heading text.
# My Markdown Document

3. Insert a blockquote by using the `>` symbol before the quoted text.
> This is a blockquote.

4. Create an unordered list with bullet points by using `-` or `*` before each item.
- Item 1
- Item 2
- Item 3

5. Add a link by enclosing the text in brackets and the URL in parentheses.
[Google](https://www.google.com/)

6. Insert an image by using `![alt text](image URL)` syntax.
![Markdown Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/208px-Markdown-mark.svg.png)

7. Format text as bold by enclosing it in `**` or `__` symbols.
This is **bold** text.

8. Create a horizontal rule by using `---` on a separate line.
---

9. Insert a code block by using triple backticks (\`) before and after the code.
