# How to format a markdown file

Format Headers:
# H1, ## H2, ### H3

Format Text:
- Bold: **Bold Text**
- Italic: *Italic Text*
- Strikethrough: ~~Strikethrough Text~~

Add Links:
[Link Text](https://www.example.com)

Insert Images:
![Image Alt Text](https://www.example.com/image.jpg)

Create Lists:
- Item 1
- Item 2
- Sublist:
   - Subitem 1
   - Subitem 2
   
Format Code:
`code`
