<a name="introduction"></a>
## Introduction
This is a sample markdown document that will serve as a template for creating a professional looking documentation. It includes a table of contents, headings, subheadings and code snippets.

<a name="installation"></a>
## Installation
To use this template, simply clone the repository and save it locally on your computer. Then, open the file in a text editor and start editing the headings and content to fit your project.

<a name="usage"></a>
## Usage
This markdown template can be used for creating any type of documentation such as user guides, API documentation, tutorials, etc. It's designed to be easy to read and navigate, making it a great option for technical documentation.

<a name="features"></a>
## Features
- Table of contents that automatically links to headings
- Easy to read and navigate layout
- Support for code snippets with syntax highlighting
- Customizable headings and subheadings
- Can be easily converted to other formats such as HTML, PDF, and Word
- Supports images and other media
