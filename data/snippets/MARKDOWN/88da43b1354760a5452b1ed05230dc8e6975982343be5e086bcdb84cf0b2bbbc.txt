## Styling Text

Here are some examples of how to style text in markdown:

- _Italicized text_ can be achieved by using one underscore or asterisk around the text.
- **Bold text** can be achieved by using two underscores or asterisks around the text.
- `Inline code` can be achieved by using backticks around the text.
- ~~Strikethrough text~~ can be achieved by using two tilde symbols around the text.
