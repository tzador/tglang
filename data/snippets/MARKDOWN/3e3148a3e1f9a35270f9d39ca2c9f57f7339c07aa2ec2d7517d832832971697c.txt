<details> 
<summary>This is a collapsible section</summary> 
  
 `Description` This is a section of code that can be expanded and collapsed, allowing for a cleaner and more organized display of information.

 `Purpose` It can be used to hide code or information that is not immediately relevant, but still needs to be included for reference or clarification.

 `Usage` To use this collapsible section, simply use the <details> tag with a <summary> tag as its child and place the content you want to hide within the <details> tag.

 `Example` 
<details>
  <summary>Example Code</summary>
