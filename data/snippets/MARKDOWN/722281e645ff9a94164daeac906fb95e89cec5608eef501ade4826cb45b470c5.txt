# Introduction
Welcome to the MARKDOWN code snippet! This code snippet will cover various features of MARKDOWN and how they can be used.

## Text Formatting
MARKDOWN supports different text formatting options including bold, italic, and strikethrough.

- *italic*
- **bold**
- ~~strikethrough~~

## Lists
Lists can be created in different ways in MARKDOWN.

### Unordered Lists
- Item 1
- Item 2
- Item 3

### Ordered Lists
1. First item
2. Second item
3. Third item

## Links
Links can be added to text in MARKDOWN using the following syntax:
[link text](url)

## Images
Images can also be added by using the following syntax:
![alt text](image_url)

## Code Blocks
MARKDOWN supports code blocks that preserve formatting and highlight syntax.
