# Markdown Tutorial
Welcome to my markdown tutorial! In this post, I will be teaching you the basics of markdown and how you can use it to create beautiful and organized documents.

## Headers
Markdown uses "#" to denote headers. A single "#" will represent a level 1 header, "##" will represent a level 2 header, and so on.

### Lists
Lists are denoted by either "-" or "*". These can represent unordered lists. To create an ordered list, simply use numbers followed by a period.

1. First ordered item
2. Second ordered item
3. Third ordered item

- Unordered list item 1
- Unordered list item 2
- Unordered list item 3

### Links
To add links in markdown, you use the format [link text](url). This will create a clickable link in your document.

[Example Link](https://www.example.com)

### Images
Images can also be added using the same format as links, except with an exclamation mark in front.

![Example Image](https://www.example.com/image.jpg)

### Code blocks
Code blocks are denoted by triple backticks ("
