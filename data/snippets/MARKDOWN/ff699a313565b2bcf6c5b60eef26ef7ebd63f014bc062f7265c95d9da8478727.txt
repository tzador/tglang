## Introduction
Markdown is a lightweight markup language used for formatting plain text. It is commonly used for creating documentation and web pages.

### Features
- Simple and easy to learn syntax
- Supports basic text formatting like bold, italic, and strikethrough
- Allows for creating headings, lists, and tables
- Can include images and links
- Can be converted into HTML for web pages
