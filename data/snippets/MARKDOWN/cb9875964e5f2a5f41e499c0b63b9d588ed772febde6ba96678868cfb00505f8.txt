Three backticks followed by a space and a backtick are used to create code blocks in Markdown. This is a long code snippet that shows the use of various features in Markdown.

Here is an example of an unordered list:
- Item 1
- Item 2
- Item 3

Here is an example of an ordered list:
1. Item A
2. Item B
3. Item C

Here is an example of a table:
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Row 1, Column 1 | Row 1, Column 2 | Row 1, Column 3 |
| Row 2, Column 1 | Row 2, Column 2 | Row 2, Column 3 |

Here is an example of formatting using emphasis:
*Italic text*
**Bold text**
***Bold and italic text***

Here is an example of a link:
[Click here to visit my website](www.example.com)

And finally, here is an example of including an image:
![Alt text](image.jpg)
