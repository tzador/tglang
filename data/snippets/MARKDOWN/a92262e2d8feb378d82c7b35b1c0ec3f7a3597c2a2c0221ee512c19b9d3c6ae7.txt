Markdown is a lightweight markup language that is used to format text on web pages. 
Modifiers such as bold, italics, and headers can be added using simple syntax. 
Lists can be created using either asterisks, plus signs, or hyphens. 
Links can be inserted using brackets and parentheses, and images using exclamation marks. 
Tables can be created using pipes and dashes.
