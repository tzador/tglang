| Syntax     | Description | 
| :----------| :----------| 
| Headers | Add columns to the table | 
| Paragraph | Add rows of content | 
| Bold        | Encloses text in double asterisks (**) | 
| Italic      | Encloses text in single asterisks (*) | 

**Sample Table:**

| Sno. | Name | Email |
|------|------|-------|
|  1   | John Doe | john@gmail.com |
| 2    | Jane Smith | jane@yahoo.com |
| 3    | Bob Johnson | bob@hotmail.com |
