###This is a heading in markdown
~~~     //triple tilde for code block
Here is a list of items:
- Item 1
    - Subitem 1.1
    - Subitem 1.2
- Item 2
    - Subitem 2.1
    - Subitem 2.2

> This is a blockquote in markdown.

Inline code: `echo "This is inline code"`  // backticks for inline code

[Link to Google](https://www.google.com/)  // markdown link syntax

**This is bold text**  // double asterisks for bold text

*This is italicized text* // single asterisk for italicized text

![Alt text for image](image.jpg)  // markdown image syntax

**Table | Header 1 | Header 2**
| ---     | ---       | ---       |
| Row 1   | Data 1    | Data 2    |
| Row 2   | Data 3    | Data 4    |

[//]: # (This is a comment in markdown.)  // markdown comment syntax
~~~
