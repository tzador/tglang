- Markdown is a lightweight markup language that is used for formatting and structuring text. It uses simple syntax and is commonly used for writing documentation and readme files.

- In markdown, the `#` symbol can be used for headings and the number of `#` symbols indicate the level of heading. For example, `# Heading 1` would create a main heading, while `## Heading 2` would create a subheading.

- Lists can be created using either `1.`, `-` or `*` symbols. Each symbol denotes a different type of list. For example, `1. Item 1` would create a numbered list, while `- Item 2` would create a bullet point list.
