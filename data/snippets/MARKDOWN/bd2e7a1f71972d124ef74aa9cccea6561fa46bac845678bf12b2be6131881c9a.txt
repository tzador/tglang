## What is Markdown?

Markdown is a lightweight markup language that is commonly used for formatting text on the web. It allows for easy formatting of text using simple syntax.

## Why use Markdown?

Markdown is widely supported and can be easily converted into other document formats, such as HTML, making it a versatile choice for writing and sharing content. It also allows for faster and more efficient writing and formatting compared to traditional word processors.

## Basic Syntax

To format text in markdown, you can use simple symbols and characters. For example, to make text bold, surround it with two asterisks like **this**. Similarly, using one asterisk around text will make it italic like *this*. Here are some other basic formatting options:

- *Italic* - *surround text with one asterisk*
- **Bold** - **surround text with two asterisks**
- ~~Strikethrough~~ - ~~surround text with two tildes~~
- `Code` - `surround text with backticks`
