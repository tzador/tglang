# This is a heading in Markdown

**Bold text** is denoted by double asterisks in Markdown.

*Italic text* can be created by using single asterisks.

`Code` can be highlighted using backticks in Markdown.

Here is a table in Markdown.

| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| Item 1   | Item 2   | Item 3   |
| Item 4   | Item 5   | Item 6   |

1. This is an ordered list item in Markdown.
2. This is another ordered list item in Markdown.
3. And another one!

- This is an unordered list item in Markdown.
- This is another unordered list item in Markdown.
- And another one!

[Link text](https://www.example.com) can be created using this format.

> Block quotes are created using the greater than symbol in Markdown.

Horizontal rules can be created using three or more hyphens.
