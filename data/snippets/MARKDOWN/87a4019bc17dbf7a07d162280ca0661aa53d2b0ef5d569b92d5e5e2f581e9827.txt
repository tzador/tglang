# Table Header
| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| Data 1   | Data 2   | Data 3   |
| Data 4   | Data 5   | Data 6   |

This is a table header with three columns and two rows, the first row being the header and the second row containing data. Each column is separated by pipes (|) and the header and data are separated by dashes (-).

To add more rows to the table, simply add another line of data using the same format.

| Data 7   | Data 8   | Data 9   |

This is a new row of data that has been added to the table. Each column still follows the same format and is separated by pipes.

You can also add formatting to your table by using the following symbols:

- * to create a bullet point
- ** to create bold text
- _ to create italic text
- ~~ to create strikethrough text
- ` to create inline code

| Column 1   | Column 2                       | Column 3                       |
| ---------- | ------------------------------ | ------------------------------ |
| * Data 1   | **Data 2**                     | _Data 3_                       |
| **Data 4** | ~~Data 5~~                     | `Data 6`                       |

This creates a more visually appealing table with different formatting options.

You can also create a table using HTML syntax within your markdown file:

<table>
  <tr>
    <th>Column 1</th>
    <th>Column 2</th>
    <th>Column 3</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
    <td>Data 3</td>
  </tr>
  <tr>
    <td>Data 4</td>
    <td>Data 5</td>
    <td>Data 6</td>
  </tr>
</table>

This creates the same table as the markdown code snippet above, but using HTML elements instead.

To add more rows to this table, you can simply add another <tr> element with the desired data.

<table>
  <tr>
    <th>Column 1</th>
    <th>Column 2</th>
    <th>Column 3</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
    <td>Data 3</td>
  </tr>
  <tr>
    <td>Data 4</td>
    <td>Data 5</td>
    <td>Data 6</td>
  </tr>
  <tr>
    <td>Data 7</td>
    <td>Data 8</td>
    <td>Data 9</td>
  </tr>
</table>

This adds a new row of data to the table using the same format as the previous rows.
