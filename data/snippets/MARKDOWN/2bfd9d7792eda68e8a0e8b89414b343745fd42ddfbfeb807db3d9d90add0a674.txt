## Advanced Markdown

1. *Emphasis* _style_ : Use asterisks or underscores for *italics* and _italics_.
2. **Strong emphasis** : Use double asterisks or underscores for **bold** and __bold__.
3. ~~Strikethrough~~ : Use double tildes for ~~strikethrough~~.
4. Headers : Use hashes for H1 to H6. # Header 1, ## Header 2, ### Header 3, #### Header 4, ##### Header 5, ###### Header 6.
5. Lists : Use * or - for unordered lists and numbers for ordered lists.
6. Links : Use [text display](link "title") for inline links and [reference][1] for reference links.
7. Images : Use ![alternative text](image link "title") for inline images and ![reference][2] for reference images.
