## Introduction
This is an annotated markdown. It showcases the use of headers, links, images, code blocks, and lists. 

## Headers

Headers are used to give structure to a document. They range from H1 to H6, with H1 being the largest.

H1 Header
===

H2 Header
---

## Links
Links can be used to reference other web pages or resources. To insert a link in a document, the following syntax is used:

[Link Text](URL)

For example, [Google Homepage](http://www.google.com) will create a link to Google's homepage.
This is particularly useful when referencing sources or providing additional information.

## Images
Images can also be inserted into a markdown document using a similar syntax as links, but with an exclamation mark before the square brackets. The syntax is as follows:

![Alt Text](Image URL)

For example, ![Markdown logo](https://upload.wikimedia.org/wikipedia/commons/4/48/Markdown-mark-white.svg) will display the markdown logo in the document.
