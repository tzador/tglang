# Introduction

This is a sample markdown document to showcase a table of contents and basic formatting in markdown.

## Purpose

This document serves as a guide for using markdown in a project. It explains the structure and features of markdown and provides examples for reference.

## Scope

This document covers the basic usage and configuration of markdown. More advanced features and customization options may not be included.

## Terminology

- **Markdown:** A lightweight markup language used for writing text that is easily converted to HTML.
- **Header:** A title or heading for a section in a markdown document.
- **Table of Contents:** A list of links that allows readers to quickly navigate to different sections of a document.

# Installation

## Requirements

To use markdown, you will need a text editor that supports markdown syntax.

## Instructions

1. Download and install a text editor that supports markdown, such as [Visual Studio Code](https://code.visualstudio.com/).
2. Create a new markdown document with a .md file extension.
3. Begin writing your markdown document using the syntax explained in this guide.

# Usage

## Basic Example

To create a header in markdown, simply prefix the text with a # symbol.

Example:

`# This is a Header`

Output:

# This is a Header

There are many advanced features in markdown, including lists, images, and tables. For a full list of available features, refer to the [official markdown documentation](https://daringfireball.net/projects/markdown/syntax).

# FAQ

## General Questions

- **Q: What is markdown?**
- A: Markdown is a lightweight markup language used for writing text that is easily converted to HTML.

- **Q: Is markdown free?**
- A: Yes, markdown is a free and open source language.

## Technical Questions

- **Q: How do I create a hyperlink in markdown?**
- A: To create a link, use the following syntax: `[link text](URL)`

- **Q: Can I include images in my markdown document?**
- A: Yes, you can use the following syntax to insert an image: `![alt text](image URL)`
