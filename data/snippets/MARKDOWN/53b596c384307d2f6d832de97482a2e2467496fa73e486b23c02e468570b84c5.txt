# Table of Contents

## Introduction

This is an example of a table of contents using headings and subheadings in Markdown. It can be used for long documents or articles to help readers navigate through the content.

## Features

- Heading 2: Used for main sections
- Heading 3: Used for sub-sections
- Ordered lists: Used for listing items in a specific order
- Unordered lists: Used for listing items without a specific order
- Links: Used for creating clickable links to other pages or websites
- Images: Used for including images in the document
- Code blocks: Used for displaying code snippets with syntax highlighting
- Bold and italic text: Used for emphasizing certain words or phrases
