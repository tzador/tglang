> "Markdown is a lightweight markup language with plain-text-formatting syntax, created in 2004 by John Gruber and Aaron Swartz. Markdown is often used for formatting readme files, for writing messages in online discussion forums, and to create rich text, using a plain text editor." (Source: Wikipedia)

## Features
Markdown has many useful features, including:
- **Simple syntax:** Markdown uses simple and intuitive syntax for text formatting, making it accessible to beginners and easy to learn.
- **Compatibility:** Markdown can be converted to HTML, which is widely supported on the internet, making it easy to share and publish documents.
- **Extensibility:** Markdown's syntax can be extended to add functionality such as tables, mathematical equations, and more.
- **Portability:** Markdown files can be edited and viewed on different platforms and devices without losing formatting.
- **Ease of use:** With its simple syntax and compatibility with text editors, Markdown allows for quick and easy creation and editing of documents.

## Basic Syntax
Some of the basic syntax in Markdown includes:
- **Headings:** Use `#` for headings, with more `#` for lower-level headings.
- **Emphasis:** Use `*` or `_` for italics, `**` or `__` for bold, and `***` or `___` for bold and italics.
- **Links:** Use `[link text](url)` to create links.
- **Lists:** Use `-` or `1.` before items for bullet and numbered lists, respectively.
