### Some Basic Markdown Syntax:

This is a *paragraph* with **bold text** and a [link](https://www.example.com). 

- Here is an unordered list:
  - First item
  - Second item
  - Third item

1. Here is an ordered list:
  1. First item
  2. Second item
  3. Third item

> This is a blockquote.
  It can span multiple lines.
