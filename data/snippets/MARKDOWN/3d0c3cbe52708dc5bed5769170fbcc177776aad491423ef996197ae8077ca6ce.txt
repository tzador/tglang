# Introduction <a name="intro"></a>

## What is Markdown? <a name="intro-1"></a>

Markdown is a lightweight markup language that is used to create formatted documents.

## Why use Markdown? <a name="intro-2"></a>

- It is simple and easy to learn
- It can be converted into HTML, making it versatile
- It is widely supported and can be used in many applications and platforms
- It allows for a clean and structured writing format
- It is great for creating documentation or note-taking

# Getting Started <a name="getting-started"></a>

## Installation <a name="getting-started-1"></a>

Markdown is typically installed through a text editor or an online converter.

## Basic Syntax <a name="getting-started-2"></a>

### Headings <a name="headings"></a>

Headings in Markdown are denoted by the '#' symbol. The number of '#' symbols used determines the heading level.

### Emphasis <a name="emphasis"></a>

Emphasis can be added to text by using '*' or '_' around the words. For bold, wrap the words in '**' or '__'. For italics, use a single '*' or '_'.

### Lists <a name="lists"></a>

Unordered lists can be created using a '-' or '*' for each item, and ordered lists can be created using numbers followed by a '.' for each item.

### Links <a name="links"></a>

Links can be added using square brackets to denote the text and parentheses to add the link URL.

### Images <a name="images"></a>

Images can be added using an exclamation mark followed by square brackets to denote the alternate text, and parentheses to add the image URL.

### Code Blocks <a name="code-blocks"></a>

Code blocks can be created using triple backticks (`) before and after the code.

### Blockquotes <a name="blockquotes"></a>

Blockquotes can be created using a '>' before the quoted text.

### Horizontal Rule <a name="horizontal-rule"></a>

A horizontal rule can be created using three or more asterisks, dashes, or underscores on a new line.

## Extended Syntax <a name="extended-syntax"></a>

### Task Lists <a name="task-lists"></a>

Task lists can be created using '- [ ]' for incomplete tasks and '- [x]' for completed tasks.

### Tables <a name="tables"></a>

Tables can be created using '|' to separate columns and '-' to create rows.

### Strikethrough <a name="strikethrough"></a>

Strikethrough text can be created using two tildes (~) before and after the text.

### Fenced Code Blocks <a name="fenced-code-blocks"></a>

Fenced code blocks can be created by using triple backticks (`) and specifying the programming language after the opening ticks.

# Tips and Tricks <a name="tips-and-tricks"></a>

## Keyboard Shortcuts <a name="keyboard-shortcuts"></a>

- Ctrl + B - Bold
- Ctrl + I - Italics
- Ctrl + L - Create Link
- Ctrl + P - Print Markdown Document
- Ctrl + K - Fenced Code Block
- Ctrl + T - Create Table
- Ctrl + Q - Blockquote
- Ctrl + H - Add Horizontal Rule

## Useful Tools <a name="useful-tools"></a>

- Markdown editors (Atom, Sublime Text)
- Online converters (Dillinger, Markdown Tables Generator)
