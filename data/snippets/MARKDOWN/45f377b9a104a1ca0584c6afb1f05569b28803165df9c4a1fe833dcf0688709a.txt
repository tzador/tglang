#### Lists
1. Unordered List
- Item 1
- Item 2
- Item 3

2. Ordered List
1. First Item
1. Second Item
1. Third Item

#### Links
To create a link, use the following syntax: [Link Text](URL)

Example: [GitHub](https://github.com/)

#### Images
To insert an image, use the following syntax: ![Alt Text](image URL)

Example: ![Markdown Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/250px-Markdown-mark.svg.png)

#### Tables
To create a table, use the following syntax:

| Column 1 | Column 2 | Column 3 |
|---|---|---|
| Row 1, Column 1 | Row 1, Column 2 | Row 1, Column 3 |
| Row 2, Column 1 | Row 2, Column 2 | Row 2, Column 3 |
