Функция СгенерироватьЧисла(НачальноеЗначение, КонечноеЗначение) Экспорт
    Переменная СписокЧисел;
	
    СписокЧисел = Новый Массив;
	
    Для i = НачальноеЗначение По КонечноеЗначение Цикл
        СписокЧисел.Добавить(i);
    КонецЦикла;
	
    Возврат СписокЧисел;
КонецФункции

Процедура ПроверитьЧисла(МассивЧисел) Экспорт
    Для Каждого Число Из МассивЧисел Цикл
        Если Число%2 = 0 Тогда
            Сообщить("Число " + Число + " четное");
        Иначе
            Сообщить("Число " + Число + " нечетное");
        КонецЕсли;
    КонецЦикла;
КонецПроцедуры

Переменная НачальноеЗначение = 1;
Переменная КонечноеЗначение = 10;

Переменная СписокЧисел = СгенерироватьЧисла(НачальноеЗначение, КонечноеЗначение);
