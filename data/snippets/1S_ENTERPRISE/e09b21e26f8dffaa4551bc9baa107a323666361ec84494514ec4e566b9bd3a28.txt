// This code snippet is an example of a 1C_ENTERPRISE program that calculates the BMI (Body Mass Index) of a person
// The program takes in the weight and height of the person and outputs their BMI

Процедура ИзмеритьBMI(Знач Вес, Знач Рост) Экспорт

// Переменные для хранения значений веса и роста
Перем ВесКг, РостМ

// Проверяем корректность введенных значений и выходим из процедуры, если они некорректные
Если Вес <= 0 Или Рост <= 0 Тогда
	Возврат;
КонецЕсли;

// Переводим вес из килограмм в грамм, а рост из метров в сантиметры
ВесКг = Вес * 1000;
РостМ = Рост * 100;

// Вычисляем BMI по формуле - вес в кг / рост в м^2
Результат = ВесКг / (РостМ * РостМ);

// Округляем результат до двух знаков после запятой
Результат = Округл(Результат, 2);

// Выводим результат в консоль
Сообщить("Ваш BMI составляет: " + Результат);

КонецПроцедуры

// Вызываем процедуру для расчета BMI для случайного человека с весом 70 кг и ростом 1.75 м
Вызвать ИзмеритьBMI(70, 1.75)

// Выводит в консоль: Ваш BMI составляет: 22.86
