// This code snippet is to calculate and display the total sales for a specific product

// Declare variables
ValueOfProduct - Number // Stores the value of the product
QuantitySold - Number // Stores the quantity of the product sold
TotalSales - Number // Stores the total sales for the product

// Prompt user for input
ValueOfProduct = Input("Enter the value of the product: ")
QuantitySold = Input("Enter the quantity sold: ")

// Calculate total sales
TotalSales = ValueOfProduct * QuantitySold

// Display result
Output("Total sales for the product: " + TotalSales) // Concatenates the string and the total sales variable for output

// End of code snippet
