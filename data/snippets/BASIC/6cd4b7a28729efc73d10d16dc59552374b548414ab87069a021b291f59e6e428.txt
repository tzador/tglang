' This code snippet is a mathematical program that calculates the area of a circle.
' It takes in the radius of the circle as an input and uses a built-in function to calculate the area.

' Input variable
DIM radius AS FLOAT

' Prompt user for radius and store in variable
INPUT "Enter radius of circle: ", radius

' Calculate area using built-in function
area = 3.14159265 * radius^2

' Print result
PRINT "The area of the circle with radius ", radius, " is: ", area

END
