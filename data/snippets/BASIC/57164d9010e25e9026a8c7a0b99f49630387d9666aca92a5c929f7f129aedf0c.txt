' This program calculates the sum of two numbers
DIM A, B, SUM ' declare integer variables A, B, and SUM
INPUT "Enter first number: ", A ' prompt user to input first number
INPUT "Enter second number: ", B ' prompt user to input second number
SUM = A + B ' calculate sum and assign it to SUM variable
PRINT "The sum of", A, "and", B, "is", SUM ' display result to user
