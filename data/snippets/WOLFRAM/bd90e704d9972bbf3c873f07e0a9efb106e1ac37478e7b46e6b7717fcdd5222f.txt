(* `Integrate` is a built-in function in Wolfram that calculates the definite integral of a given function over a range of values *)

Integrate[Exp[-x^2],{x,0,1}]

(* The code above integrates the function `Exp[-x^2]` over the range from 0 to 1 *)

Output: `1/2 Sqrt[\[Pi]] Erf[1]` (* This is the result of the definite integral *)
