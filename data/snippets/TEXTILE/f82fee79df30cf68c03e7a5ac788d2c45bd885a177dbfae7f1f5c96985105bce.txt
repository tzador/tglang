h1. Welcome to my textile website!

p. This is a paragraph of text in Textile format. It can contain *bold*, _italic_, and even @code@ formatting.

bq. This is a block quote in Textile. You can add multiple lines of text and it will be automatically formatted.

ol. This is an ordered list in Textile. You can add items in a numbered order using the hash symbol.

# List item one
# List item two
# List item three

ul. This is an unordered list in Textile. Items will be denoted by asterisks.

* List item one
* List item two
* List item three

h2. Textile Table

|_. Column 1 |_. Column 2 |_. Column 3 |
| Item 1     | Item 2     | Item 3     |
| Item 4     | Item 5     | Item 6     |
| Item 7     | Item 8     | Item 9     |

p. This is another paragraph in Textile. You can add links using double square brackets and specify the link and text in parenthesis.

"Link to Google":http://www.google.com (Google)

p. You can also add images in Textile using exclamation marks and parenthesis to specify the image URL and alt text.

!http://www.example.com/image.jpg(Image Caption)!

pre. This is a preformatted block in Textile.
You can add code snippets with optional language formatting by specifying the language after the double colons.

pre::python.
print("Hello World")
