1. *Bold text*         // Output: <b>Bold text</b>
2. _Italic text_       // Output: <i>Italic text</i>
3. *#Header#*          // Output: <h1>Header</h1>
4. [Link text|www.example.com]  // Output: <a href="www.example.com">Link text</a>
5. - List item 1       // Output: <ul><li>List item 1</li></ul>
6. - List item 2       // Output:              <li>List item 2</li>
7. - List item 3       // Output:              <li>List item 3</li></ul>
8. !ImageAltText(www.example.com/image.jpg) // Output: <img src="www.example.com/image.jpg" alt="ImageAltText">
9. |_. Header 1 |_. Header 2 | // Output: <table><tr><th>Header 1</th><th>Header 2</th></tr>
10. |Cell 1 | Cell 2 |         // Output: <tr><td>Cell 1</td><td>Cell 2</td></tr></table>
