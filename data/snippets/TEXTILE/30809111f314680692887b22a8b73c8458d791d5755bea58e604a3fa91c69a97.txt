h3. How to use Textile for styling:

ul. 
* Use "*" for bold text
# Use "#" for numbered list
_ Use "_" for italicized text
- Use "-" for bulleted list
\[link\|Text\] for creating a link

p. It's that easy!

p. Textile is perfect for creating simple yet professional-looking web content.

h3. Some examples:

p. Here's an example of how to create a table in Textile:

|_. Name |_. Age
|John |28
|Emily |30
|Mark |25

p. And here's how to insert an image:

!https://www.example.com/images/logo.png!

p. These are just a few examples of Textile's capabilities. You can explore more formatting options in the official documentation.
