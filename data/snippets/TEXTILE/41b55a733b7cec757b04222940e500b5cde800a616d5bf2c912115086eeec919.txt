# Trying out the textile markup language

p. Textile is a simple markup language that can be used to format text in a visually appealing manner.
p. It is commonly used in web development and blogging platforms as a way to easily style content.

p. Here's an example of a formatted list using textile:
* Item 1
* Item 2
* Item 3

h2. Textile can also be used to add images to a document:

!https://example.com/image.jpg!

p. Textile also supports tables, which can be useful for displaying data in an organized manner. Here's an example of a basic table:

|_. Name |_. Age |_. Occupation |
| John | 28 | Developer |
| Sarah | 32 | Designer |

p. Textile also has the ability to create links, making it easy to add clickable text to a document. Here's an example of a link:

"A simple link to Google":https://www.google.com

p. Textile also has some advanced formatting options, such as making text bold, italicized, or underlined:

p. *This text is bold*
p. _This text is italicized_
p. +This text is underlined+

p. Lastly, Textile has the ability to add blockquotes, perfect for highlighting important information or quotes:

bq. "The only way to do great work is to love what you do." - Steve Jobs

p. Overall, Textile is a powerful and versatile markup language that can be used in a variety of ways to enhance the presentation of text. Give it a try and see what you can create!
