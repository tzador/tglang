bq. This is a blockquote in Textile. It can be used to highlight important information or quotes from other sources.

h2. Textile Formatting

p. Textile allows you to easily format your text. For example, _this text will be italic_, *this text will be bold*, and +this text will be underlined+.

p. You can also create links in Textile using the "link text":http://website.com format.

p. Images can be inserted using the !image_url! format.

h2. Tables in Textile

|_. Column 1 |_. Column 2 |_. Column 3 |

| Row 1, Column 1 | Row 1, Column 2 | Row 1, Column 3 |
| Row 2, Column 1 | Row 2, Column 2 | Row 2, Column 3 |
| Row 3, Column 1 | Row 3, Column 2 | Row 3, Column 3 |

h2. Textile Block Commands

bc. This is a code block in Textile. It can be used to display code snippets or blocks of code.

pre. This is a preformatted block in Textile. It preserves all line breaks and white spaces.

h2. Textile Inline Commands

@red. This text will be displayed in red color.

@size(18pt). This text will have a font size of 18 points.

@link(#). This text will be linked to the specified URL.

@target(_blank). This text will open the link in a new tab.
