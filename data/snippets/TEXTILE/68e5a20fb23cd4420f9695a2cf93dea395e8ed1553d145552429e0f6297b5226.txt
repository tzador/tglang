h1. *Welcome* to _TEXTILE_

h2. *About* this language
p(#info). Textile is a markup language that allows for easy formatting of text and documents. It is similar to HTML, but with a simpler syntax that is easier to learn and use.

h2. *Syntax* examples
p(#info). Textile uses symbols and tags to format text. Here are some examples:
- "*" creates bold text
- "_" creates italicized text
- "#" creates a numbered list
- "*" creates a bulleted list

h2. *Advanced* formatting
p(#info). Textile also allows for more complex formatting, such as:
- Links: "This is a link":http://www.example.com
- Images: "This is an image":http://www.example.com/image.jpg
- Blockquotes:
bq. This is a blockquote

h2. *Tables*
p(#info). Textile can also create tables with the following syntax:
|_. Header 1 |_. Header 2 |_. Header 3 |
| Cell 1 | Cell 2 | Cell 3 |
| Cell 4 | Cell 5 | Cell 6 |

h2. *Code* examples
p(#info). Textile can also display code snippets with syntax highlighting. For example:
bc.
