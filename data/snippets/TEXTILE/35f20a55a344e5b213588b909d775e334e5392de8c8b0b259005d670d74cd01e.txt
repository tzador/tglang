p. Hello, this is a sample of *Textile* code snippet.

p. To create a table in Textile, use the following syntax:

|_. Column 1 |_. Column 2 |_. Column 3 |
| Cell 1 | Cell 2 | Cell 3 |
| Cell 4 | Cell 5 | Cell 6 |

p. This will create a table with 3 columns and 2 rows.

p. To add styling to your table, use the following syntax:

|=. main_table
|_. Header 1 |_. Header 2 |_. Header 3 |
| | Cell 1 | Cell 2 | Cell 3 |
| | Cell 4 | Cell 5 | Cell 6 |

p. This will apply the "main_table" class to the table and center the headers.

p. To add a link in Textile, use the following syntax:

"This is a link to Google":http://www.google.com

p. To add an image in Textile, use the following syntax:

!https://www.example.com/image.jpg!

p. To create a list in Textile, use the following syntax:

# Item 1
# Item 2
# Item 3

p. This will create a numbered list.

p. To add emphasis to text, use the following syntax:

p<. *Bold text*
p<. _Italic text_
