// This code snippet in TEXTILE language is an example of creating a table with headings and data rows

|_. Field 1 |_. Field 2 |_. Field 3 |
| Data 1 | Data 2 | Data 3 |
| Data 4 | Data 5 | Data 6 |
| Data 7 | Data 8 | Data 9 |

//Explanation: This code creates a table with three columns (Field 1, Field 2, Field 3) and three rows of data. The heading for each column is denoted by |_. and the data cells are denoted by |. Two pipes (|) separate each cell, while a period (.) is used to denote the start of a new row.
