h1. **Welcome to My Website!**

p. This is a *sample* textile code snippet that shows how to style your website using *Textile* markup language.

p. Textile is an easy-to-use markup language that allows you to create simple and elegant web pages. It offers a set of simple and intuitive syntax for formatting text, images, links, and more.

h2. *Basic Syntax*

p. The basic syntax for styling text is simple. You can use asterisks to create emphasis, underscores to create underlines, and double quotes to create citations. For example:

_This is underlined text_ *This is emphasized text* "This is a citation"

h2. *Lists*

p. Textile also offers a variety of options for creating lists. You can use the "*" bullet point symbol for unordered lists and the "#" number symbol for ordered lists. Nested lists can also be created by indenting using tabs. For example:

* Item 1
* Item 2
** Nested item 1
** Nested item 2
** Nested item 3

h2. *Links and Images*

p. Textile provides an easy way to insert links and images into your webpage. To add a link, use the "link text":http://linkurl syntax. To insert an image, use the "image alt text":imageurl syntax. For example:

"This is a link to Google":http://www.google.com
"This is an image of a cat":cat.jpg

h2. *Tables*

p. Tables can also be created using Textile. Use the "|" symbol to separate columns and "-" to create headers. For example:

|_. Column 1 |_. Column 2 |_. Column 3 |
| Row 1 | Row 1 | Row 1 |
| Row 2 | Row 2 | Row 2 |

h2. *Advanced Styling*

p. Textile also offers more advanced styling options, such as creating blocks of code, inline code, and block quotes. For example:

bc. This is a block of code.

@This is inline code@

bq. This is a block quote.

h2. *Final Thoughts*

p. Textile may seem daunting at first, but once you get the hang of it, it is a powerful and efficient tool for styling your website. With its simple and intuitive syntax, Textile makes it easy to create beautiful and functional web pages. Give it a try and see for yourself!
