# Class Definition

h2. Company
The Company class is used to represent a business organization.

|_. Property |_. Type |_. Description |
| Name        | String  | The name of the company |
| Industry    | String  | The industry the company operates in |
| Members     | List    | A list of employees |

h3. Method: add_member
Adds a new employee to the Members list.

|_. Parameter |_. Type |_. Description |
| Name         | String | The name of the employee |
| Role         | String | The role of the employee |
