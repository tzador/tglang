h1. Textile Example
* Intro
Welcome to my "Textile" example. Textile is a markup language that allows for simple and easy formatting of text.

* Lists
- Item 1
- Item 2
- Item 3

# Nested List 2
## Nested Item 1
## Nested Item 2
## Nested Item 3

* Blockquote
bq. This is a blockquote. Textile also allows for blockquotes to be nested within one another.

* Images

* Bold and Italic

* Code snippet

color: #333;
background: #eee;
padding: 10px;
@

* Tables
|_. Column 1 |_. Column 2 |_. Column 3
| Cell 1 | Cell 2 | Cell 3 |
| Cell 4 | Cell 5 | Cell 6 |

* Footnote
fn1. This is a footnote. Textile allows for footnotes to be added at the bottom of a document.
