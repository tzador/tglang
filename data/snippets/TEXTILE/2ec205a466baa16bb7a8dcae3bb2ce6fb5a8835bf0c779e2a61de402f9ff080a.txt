-> TEXTILE Example Code:

h1. Hello, World!
<p>This is an example of text in <em>Textile</em> markup language.</p>
<p>You can format your text in <strong>bold</strong>, <em>italic</em>, or <del>strikethrough</del> using corresponding symbols.</p>
<p>You can also add links using the <code>"link text":http://example.com</code> syntax.</p>
<p>Lists are easy to create:</p>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<p>You can also create ordered lists using numbers:</p>
# Ordered list item 1
# Ordered list item 2
# Ordered list item 3
<p>It's easy to add images as well:</p>
<image>http://example.com/image.jpg</image>
<p>You can add colors to your text using the <code>%{color:red}red text%</code> syntax.</p>
<p>Tables can be created using vertical bars and hyphens:</p>
|_. Column 1 |_. Column 2 |
| Value 1 | Value 2 |
| Value 3 | Value 4 |
