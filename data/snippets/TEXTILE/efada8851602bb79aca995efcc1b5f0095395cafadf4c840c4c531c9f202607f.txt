h1(#header). Title   // creates a h1 heading with the id "header"
p{color:red}. This is a paragraph with red text  // creates a paragraph with red text
*# This is a list item with a checked checkbox    // creates a list item with a checked checkbox
* This is a list item with an unchecked checkbox  // creates a list item with an unchecked checkbox
!https://example.com/image.jpg!  // inserts an image from a URL
|_. Column 1 |_. Column 2 |  // creates a table with two columns using the table header tag
|<. Left aligned |>. Right aligned |  // aligns the text in the table cells
|<. Left aligned |
|>. Right aligned |
bc{background:#FFFF00}. Background color  // creates a block of code with a yellow background
fn1(font-size:18px). This text has a font-size of 18px // changes the font size of the text
bq.. This is a block quote // creates a block quote
#footer(text-align:center). Footer // creates a footer with centered text
