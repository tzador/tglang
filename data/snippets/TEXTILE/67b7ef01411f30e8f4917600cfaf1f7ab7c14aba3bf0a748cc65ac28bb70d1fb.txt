h1. Welcome to my Textile Code!

p. *Textile* is a markup language that allows for simple and straightforward formatting of text. It is commonly used for creating documents and web pages.

bq. This is a blockquote. It is used to highlight important information or quotes from other sources.

bc.
div. This is a divider, which can be used to create a visual break between sections.

pre. This is a preformatted block. It is used to display code snippets or other text exactly as it is written, without any formatting.

ul. This is an unordered list, used to create a bulleted list of items.

li. This is a list item, used within the ul tag to indicate the items in the list.

h2. Examples of Textile Formatting:

p. Here is some *bold* and _italic_ text.

p(#red). This is a paragraph with a red background color.

p[align="center"]. This is a centered paragraph.

p>. This is a paragraph with a right-aligned border.

p<. This is a paragraph with a left-aligned border.

p=. This is a paragraph with a centered border.

table.
|_. Column 1 |_. Column 2 |_. Column 3 |
| Row 1, Column 1 | Row 1, Column 2 | Row 1, Column 3 |
| Row 2, Column 1 | Row 2, Column 2 | Row 2, Column 3 |
| Row 3, Column 1 | Row 3, Column 2 | Row 3, Column 3 |
This is a table with 3 columns and 3 rows.
