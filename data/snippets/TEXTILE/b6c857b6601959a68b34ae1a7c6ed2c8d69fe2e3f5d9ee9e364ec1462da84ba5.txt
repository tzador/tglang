h1. Welcome to My Website

p. This is a sample page written in Textile markup language.

h2. Introduction

p. Textile is a simple yet powerful markup language used for formatting text on web pages. It is easy to learn and use, making it a popular choice among web developers.

h3. Basic Formatting

p. Textile allows you to easily format text with just a few keystrokes. For example, to make text bold, you simply surround it with asterisks like this: *bold text*.

p. You can also make text italic by placing it between two underscores, like this: _italic text_.

h4. Lists

p. Textile also supports creating lists. You can create unordered lists by placing an asterisk before each item, like this:

* First item
* Second item
* Third item

p. Ordered lists can be created by placing a number followed by a period before each item, as shown below:

# First item
# Second item
# Third item

h5. Links

p. Links can easily be created in Textile by placing the link text in quotes, followed by a colon, and then the URL. For example:

"Textile Markup Language":https://www.textile-lang.com

p. You can also add a title to your link by placing it in parenthesis after the URL, like this:

"Textile Markup Language (Learn More)":https://www.textile-lang.com

h6. Images

p. Adding images to your web page is also made simple with Textile. You can insert an image by placing an exclamation point, followed by the image URL, and then an exclamation point again. For example:

!https://www.imageurl.com/image.jpg!

p. You can also add a title to your image by placing it in square brackets after the image URL, like this:

!https://www.imageurl.com/image.jpg(Image Title)!

h7. Tables

p. Textile also has support for creating tables. You can create a table by using a combination of pipes and dashes, like this:

|_. Column 1 |_. Column 2 |_. Column 3 |
| Cell 1 | Cell 2 | Cell 3 |
| Cell 4 | Cell 5 | Cell 6 |
