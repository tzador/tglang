h2. *Welcome to my Website*

Welcome to my website! This is a sample page written in Textile, a simple markup language for formatting text. Below are some examples of how to use Textile to make your website look great.

h3. *Headings*
You can use various levels of headings to organize your page. Here are a few examples:

h1. Heading 1
h2. Heading 2
h3. Heading 3
h4. Heading 4

h3. *Text Formatting*
Textile offers different ways to format your text. Let's take a look at some of them:

bq. "Bold text"
_Italic text_
"Underlined text"
*@ Highlighted text @*

h3. *Lists*
Textile supports both ordered and unordered lists. Here's how to create them:

# First item
# Second item
# Third item

* First item
* Second item
* Third item

h3. *Links*
Creating links in Textile is simple. Here are a few examples:

"Facebook":https://www.facebook.com
"Twitter":https://twitter.com

h3. *Images*
Images can also be added to your webpage using Textile:

!http://example.com/image.jpg!

h3. *Tables*
Tables are a great way to organize data on your webpage. Here's how to create a basic table in Textile:

|_. Column 1 |_. Column 2 |_. Column 3 |
| Row 1      | Row 1      | Row 1      |
| Row 2      | Row 2      | Row 2      |
| Row 3      | Row 3      | Row 3      |

h3. *Code Blocks*
If you need to display code on your webpage, you can use Textile's code blocks:

bc. "public static void main(String[] args) {
    System.out.println("Hello, world!");
}"

h3. *Footnotes*
You can also add footnotes in Textile. Here's an example:

This is a sentence with a footnote.^[This is a footnote.]

h3. *Blockquotes*
Blockquotes are a useful way to highlight important text:

bq.. "Here is a long blockquote that I want to highlight and display prominently on my webpage."
