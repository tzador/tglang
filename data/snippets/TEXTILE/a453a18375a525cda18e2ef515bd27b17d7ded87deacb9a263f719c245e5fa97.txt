# Heading
This is an example of **bold** and _italic_ text in *TEXTILE* language.

p{color:red}. This is a paragraph with red text

h2{color:blue}. This is a second level heading with blue text

ul
* This is a bulleted list item
* Another list item
** This is a nested list item
* Third list item

bq. This is a blockquote
p[title="This is a tooltip"]. This is a paragraph with a tooltip
