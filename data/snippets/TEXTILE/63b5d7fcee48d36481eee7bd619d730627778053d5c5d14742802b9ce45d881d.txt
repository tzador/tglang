.What is TEXTILE
TEXTILE is a plain text markup language that is used to format and style text for web-based content.
It uses a simple syntax that is easy to read and write. Using TEXTILE, you can change the appearance of text by adding different markup tags and attributes.

Some example TEXTILE tags include:

* * for bold text
* _ for italic text
* ^ for superscript text
* ~ for subscript text
