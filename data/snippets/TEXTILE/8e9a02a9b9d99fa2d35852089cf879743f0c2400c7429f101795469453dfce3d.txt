<h1>Welcome to My Website!</h1> 
<h2>This is a subheading in Textile markup language</h2> 
<p>Browsing this website is a breeze with Textile's simple and clean syntax.</p> 
<p><em>Emphasize any important text by wrapping it in asterisks.</em></p> 
<blockquote><p>Blockquotes can also be easily created in Textile, just use a greater than sign at the beginning of each line.</p></blockquote> 

<p>Below is an example list:</p> 
<ul> 
  <li>Item 1</li> 
  <li>Item 2</li> 
  <li>Item 3</li> 
  <li>Item 4</li> 
</ul> 

<p>And here is an example table:</p> 
<table> 
  <thead> 
    <tr> 
      <th>Column 1</th> 
      <th>Column 2</th> 
      <th>Column 3</th> 
    </tr> 
  </thead> 
  <tbody> 
    <tr> 
      <td>Row 1, Column 1</td> 
      <td>Row 1, Column 2</td> 
      <td>Row 1, Column 3</td> 
    </tr> 
    <tr> 
      <td>Row 2, Column 1</td> 
      <td>Row 2, Column 2</td> 
      <td>Row 2, Column 3</td> 
    </tr> 
  </tbody> 
</table>

<p>Textile also has support for linking to external websites, like this: <a href="https://www.example.com/">Click here to visit my website</a>.</p>

<p>Lastly, we can add images to our website using the following syntax:</p>
<img src="https://example.com/image.jpg" alt="Example Image" title="This is an example image">

<p>Thanks for checking out my website, I hope you enjoyed it!</p>
