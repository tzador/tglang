# Textile Markup Example

h1. Heading 1
p. This is a paragraph of text. It can be formatted using *bold*, _italic_, and +underline+.

h2. Heading 2
p. Another paragraph here, with a %{color:red}red text% and a %{font-size:20px}large font%.

h3. Lists and Links
p. * Bullet list
* Second item
** Nested bullet
# Numbered list
# Second item
p. This is a _link_: "Google":http://www.google.com

h4. Tables
|_. Column 1 |_. Column 2 |_. Column 3 |
| Cell 1 | Cell 2 | Cell 3 |
| Cell 4 | Cell 5 | Cell 6 |
| Cell 7 | Cell 8 | Cell 9 |

h5. Images
p. This is an image: !http://www.example.com/image.jpg!.

h6. Code Snippets
bc. Console.log("Hello World");
p. This snippet shows how to print "Hello World" in the console using JavaScript.

h7. Divs and Classes
div(class="container")
p. This text is inside a div with a class of "container".
