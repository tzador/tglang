p. My website is coded using Textile markup, which makes it easy to style and format content.

Code block:
bc. <h1>Welcome to my website!</h1>
<p>This is an example of a code block using Textile markup.</p>

p. Images can also be inserted using Textile:
!http://example.com/image.png!

p. To add a caption below the image, use the following format:
!http://example.com/image.png!: This is the image caption.

p. You can also add links to images:
!http://example.com/image.png!:http://www.example.com
