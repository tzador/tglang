h1. Welcome to my Textile Code!
This is a paragraph that explains the purpose of my code.

p. I am using the Textile markup language to create a complex and long code snippet.
Let's start with some basic formatting.

h2. Text Formatting
*bold* - to make text bold
_italic_ - to make text italicized
+inserted+ - to make text underlined
-strikethrough- - to make text strikethrough
^superscript^ - to make text superscript
~subscript~ - to make text subscript
%span{color:red}red text% - to color text

h2. Links and Images
"Link Text":https://www.example.com - to create a link
!{width: 200px}(image.jpg) - to insert an image with specified width

h2. Lists
* Unordered list item 1
** Unordered list item 1.1
# Ordered list item 1
## Ordered list item 1.1

h2. Blockquote and Code
bq. This is a blockquote.

bc. Code snippet {
    language: TEXTILE,
    type: long and complex
}

h2. Tables
|_. Column 1 |_. Column 2 |
| Row 1, Col 1 | Row 1, Col 2 |
| Row 2, Col 1 | Row 2, Col 2 |

h2. Extended Formatting
"h3. Headings":# - for headings up to h6
*# Enumerated list item 1 - to create a numbered list
??highlighted text?? - to add background color to text
+<ins>inserted with style</ins>+ - to add custom CSS styles to text
-"blockquote":http://www.example.com - to link a blockquote

h2. Advanced Elements
p(quote)._ This is a paragraph quote.
The line above is used to create a paragraph quote, which can be styled using CSS.

h3. Notes
Note. This is a note element.
Note information can be added here for readers to take note of.

!!Warning!!
This is a warning element.
In case of any warnings, relevant information should be placed here.
