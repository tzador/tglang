h1. Welcome to the Textile Programming Language!

bq. Textile is a simple markup language that can be used to format text and create complex layouts. It is designed to be easy to read and write, and is often used in wiki applications and content management systems.

h2. Basic Formatting

*This text will be italic*
_This text will also be italicized_

**This text will be bold**
__This text will also be bolded__

h3. Lists

# This is a numbered list item
# This is another numbered list item

* This is a bulleted list item
* This is another bulleted list item

h4. Links

"Google":http://www.google.com
"Bing":http://www.bing.com

h6. Tables

|_. Column 1 |_. Column 2 |
| Row 1, Column 1 | Row 1, Column 2 |
| Row 2, Column 1 | Row 2, Column 2 |

h7. Advanced Formatting

p(. This is a paragraph with a class.
p(#id). This is a paragraph with an ID.
