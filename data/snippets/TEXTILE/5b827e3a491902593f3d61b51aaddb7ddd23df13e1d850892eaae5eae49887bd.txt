*# Text Formatting

p. Textile offers a variety of formatting options for your text, including *bold*, _italic_, and -strikethrough-.

+h2. Lists

# Unordered List

* First item
* Second item
* Third item

# Ordered List

# First item
# Second item
# Third item

+h2. Links

p. You can create links to other websites by using the following syntax:

"Link text":http://www.example.com

+h2. Images

p. Images can also be embedded in a Textile document using the following syntax:

!image.jpg!

+p. You can also add a title and a caption to your images:

!image.jpg!:title. Example Image|caption. This is an example image.

+h2. Tables

p. Tables can be easily created in Textile by using the following syntax:

|_. Column 1 |_. Column 2 |_. Column 3 |
| Row 1 | Cell 1 | Cell 2 |
| Row 2 | Cell 3 | Cell 4 |

+h2. Code Snippets

p. Code snippets can be added to a Textile document using the following syntax:

@code snippet@

+p. You can also specify the language of the code snippet for syntax highlighting:

@code snippet@python@
