---						// This creates a horizontal rule
![Image](image.jpg)		// This adds an image
p(#id). Lorem ipsum dolor sit amet, _consectetur_ adipiscing elit.		// This creates a paragraph with an id and italicized text
bq. This is a block quote.		// This creates a block quote
|===					// This creates a table
|Column 1 | Column 2 |		// This defines the columns in the table
|Cell 1 | Cell 2 |			// This adds content to the first row
|Cell 3 | Cell 4 |			// This adds content to the second row
|===						// End of table
@button[Click me]{url}		// This creates a button with a link
#Heading 1					// This creates a heading 1
* List item 1				// This creates a bullet point list item
* List item 2				// This creates another bullet point list item
** Nested list item 1		// This creates a nested bullet point list item
** Nested list item 2		// This creates another nested bullet point list item
# List item 3				// This creates a number list item
## List item 3.1			// This creates a sub-number list item
## List item 3.2			// This creates another sub-number list item
h4. This is a level 4 heading		// This creates a heading 4
bq.. This is a nested block quote.		// This creates a nested block quote with double full stops
|_. Column 1 |_. Column 2 |		// This creates a title row for a table
|\\| CellA | CellB |			// This creates a table row with escaped characters
|\* | CellC | CellD |			// This creates another table row with escaped characters
\* Note: Escape character is backslash ('\'')		// This is a comment
