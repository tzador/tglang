h1. Welcome to Textile

p. Textile is a lightweight markup language that allows you to easily format text and generate HTML code. It's simple syntax makes it easy to learn and use for creating rich text content.

bq. Textile was created in 2002 by "Dean Allen":https://deanish.com/. Since then, it has gained popularity and is used by many websites, including "GitHub":https://github.com/.

h2. Basic Text Formatting

p. Textile allows you to format text using simple markup symbols. For example, *bold*, _italic_, and +underline+ syntax is supported.
