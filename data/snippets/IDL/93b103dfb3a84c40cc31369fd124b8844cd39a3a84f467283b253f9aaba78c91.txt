pro calculate_magnitude, data
; This function calculates the magnitude of a 3-dimensional vector
; Inputs: 
;  - data: a 3-element array with the x, y, and z components of the vector
;
; Output:
;  - magnitude: a scalar representing the magnitude of the vector
;
; Example:
;  data = [1.2, 3.4, 5.6]  ; Vector with x = 1.2, y = 3.4, z = 5.6
;  magnitude = sqrt(data[0]^2 + data[1]^2 + data[2]^2) ; Calculate magnitude
;
; Return the magnitude value
return magnitude
end
