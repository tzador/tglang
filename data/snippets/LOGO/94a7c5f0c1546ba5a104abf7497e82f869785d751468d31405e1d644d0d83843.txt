TO square :length
REPEAT 4 [ FD :length RT 90 ]
END

Explanation:
- TO square :length: This line creates a procedure named "square" that takes in a parameter named "length".
- REPEAT 4 [ FD :length RT 90 ]: This line repeats the commands within the brackets 4 times, which are to move forward by the specified length and then turn right by 90 degrees.
- END: This line marks the end of the procedure.
- An example usage of this procedure would be: `square 50`, which would draw a square with each side being 50 units long.
