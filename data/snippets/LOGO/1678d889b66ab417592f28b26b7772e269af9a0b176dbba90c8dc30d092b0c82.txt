to star :size
  repeat 5 [fd :size
            rt 144]

; This is a function called "star" that takes in a parameter called "size"
; It repeats the instructions within the brackets 5 times
; The instructions include moving forward (fd) by the specified size
; and turning to the right (rt) by 144 degrees each time
; This function can be called multiple times with different size values to create different sized stars.
