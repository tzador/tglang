' This is a comment
Dim a, b ' Declaring variables a and b
a = 10 ' Assigning value 10 to variable a
b = 20 ' Assigning value 20 to variable b
If a > b Then ' If statement to check if a is greater than b
    MsgBox "a is greater than b" ' If condition is true, display message box
ElseIf a < b Then ' ElseIf statement to check if a is less than b
    MsgBox "b is greater than a" ' If condition is true, display message box
Else ' Else statement executed if both conditions are false
    MsgBox "a is equal to b" ' If no conditions are met, display message box
End If ' End of If statement
