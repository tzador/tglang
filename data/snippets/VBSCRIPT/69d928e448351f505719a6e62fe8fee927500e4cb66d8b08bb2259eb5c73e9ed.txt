' This is a comment
Option Explicit ' Requires all variables to be declared before use
Dim num1, num2 ' Declaring variables for storing numbers
num1 = 10 ' Assigning value to num1 variable
num2 = 20 ' Assigning value to num2 variable
' Here, we will calculate the sum of num1 and num2
Dim sum ' Declaring a variable to store the sum
sum = num1 + num2 ' Calculating the sum and assigning it to the sum variable
Msgbox "The sum of " & num1 & " and " & num2 & " is " & sum ' Displaying a message box with the result
' Output: "The sum of 10 and 20 is 30"
