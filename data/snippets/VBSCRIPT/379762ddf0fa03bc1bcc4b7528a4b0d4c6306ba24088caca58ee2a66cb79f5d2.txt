'This is a conditional statement using IF, ELSEIF, and ELSE

    Response.Write("X is a positive number.")
ElseIf x < 0 Then 'if x is less than 0
    Response.Write("X is a negative number.")
Else 'if x is equal to 0
    Response.Write("X is zero.")
End If 'end of conditional statements
