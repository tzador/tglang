'This code snippet in VBScript will prompt the user for their name, and greet them with a personalized message!
Dim name 'creates a variable to store the user's name
name = InputBox("Please enter your name:") 'prompts the user for their name and stores it in the 'name' variable
msgBox "Hello, " & name & ". Nice to meet you!" 'greets the user with a personalized message using the 'name' variable
