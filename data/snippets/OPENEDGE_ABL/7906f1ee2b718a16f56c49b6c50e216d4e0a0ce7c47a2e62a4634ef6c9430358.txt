DEFINE VARIABLE i AS INTEGER NO-UNDO.
DO i = 1 TO 10:
    DISPLAY i SKIP.
END.
/* The above code creates an INTEGER variable named 'i' with NO-UNDO option.
It then runs a DO loop from 1 to 10, displaying the current value of i on each iteration and skipping a line after each display.
The loop ends with an END statement. */
