\documentclass{article} % defining document class
\usepackage{amsmath} % importing amsmath package
\begin{document} % starting document environment
\title{My First Document} % setting document title
\date{\today} % setting date to today's date
\maketitle % generating title based on title and date
\section{Introduction} % adding a new section
This is my first document using LaTeX. % adding text
In this document, I will be exploring the basics of LaTeX. % adding more text
\subsection{Formatting} % adding a subsection
LaTeX has various built-in formatting options, such as \textit{italics}, \textbf{bold}, and \underline{underline}. % formatting text using built-in commands
\subsection{Equations} % adding another subsection
LaTeX also has powerful math typesetting capabilities. % adding text
I can write equations using the amsmath package, such as the famous Pythagorean theorem: % adding description
\begin{equation} % starting equation environment
a^2 + b^2 = c^2 % writing equation
\end{equation} % ending equation environment
\end{document} % ending document environment
