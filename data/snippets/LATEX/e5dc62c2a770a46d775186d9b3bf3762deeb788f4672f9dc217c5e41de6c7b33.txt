\documentclass{article} % Defines the type of document to be created, in this case an 'article'
\usepackage[utf8]{inputenc} % Allows the use of special characters in the document
\title{My Example Document} % Sets the title of the document
\author{John Smith} % Sets the author of the document
\date{\today} % Sets the date of the document to be the current date

\begin{document} % Begins the document

\maketitle % Creates a title based on the information provided in the preamble (title, author, date)

\section{Introduction} % Creates a section heading
This is an example document showcasing the use of \LaTeX{} for document creation.

\subsection{Purpose} % Creates a subsection heading within the 'Introduction' section
The purpose of this document is to demonstrate the capabilities of \LaTeX{} for creating professional-looking documents with ease.

\section{Main Content} % Creates a section heading for the main content of the document
\LaTeX{} provides a wide range of features for formatting text, equations, figures, and tables. It also allows for easy referencing and cross-referencing within the document.
