\section{Introduction} 
\LaTeX is a powerful typesetting language that is widely used in academic and scientific documents. It allows for precise control of formatting, making it ideal for creating professional documents. In this code snippet, we will explore some of the different typesetting features of \LaTeX, such as font styles, lists, and equations. 
\subsection{Font styles} 
There are three different font styles in \LaTeX: \textbf{bold}, \textit{italic}, and \texttt{typewriter}. These can be easily applied using the corresponding commands: \textbackslash textbf\{...\}, \textbackslash textit\{...\}, and \textbackslash texttt\{...\}. 
\subsection{Lists} 
Creating lists in \LaTeX is also simple. We can use the \textbackslash begin{itemize} and \textbackslash end{itemize} commands to create a bulleted list: 
\begin{itemize} 
\item First item 
\item Second item 
\item Third item 
\end{itemize} 
We can also use the \textbackslash begin{enumerate} and \textbackslash end{enumerate} commands for a numbered list: 
\begin{enumerate} 
\item First item 
\item Second item 
\item Third item 
\end{enumerate} 
\subsection{Equations} 
One of the most powerful features of \LaTeX is its ability to typeset complex equations. We can use the \textbackslash begin{equation} and \textbackslash end{equation} commands to create an equation environment, and then use various symbols and commands to input the equation: 
\begin{equation} 
e^{i\pi} + 1 = 0 
\end{equation} 
In this example, we use the \textbackslash pi command to display the Greek letter pi and the \textbackslash times command to show multiplication. We can also use the \textbackslash frac command to create fractions within equations: 
\begin{equation} 
\frac{d}{dx}(x^2) = 2x 
\end{equation}
