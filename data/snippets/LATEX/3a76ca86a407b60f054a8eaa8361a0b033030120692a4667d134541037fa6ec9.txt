% This is a comment in LATEX.

\documentclass{article} % Declaring the document class as 'article'.
\usepackage[utf8]{inputenc} % Using the utf8 encoding for input.
\title{My First LATEX Document} % Setting the title of the document.
\author{John Doe} % Setting the author of the document.

\begin{document} % Starting the document environment.

\maketitle % Adding the title to the document.

\section{Introduction} % Starting a new section.

This is my first LATEX document. % Adding text to the document.

\subsection{Background} % Starting a subsection.

LATEX is a markup language used for typesetting documents. % Adding more text.

\begin{itemize} % Starting an itemize environment.

    \item LATEX is widely used in academia for writing research papers % Adding an item.
    and articles.

    \item It offers a professional and consistent layout for documents. % Adding another item.

\end{itemize} % Ending the itemize environment.

\subsection{Features} % Starting another subsection.

Some of the features of LATEX include: % Adding more text.

\begin{enumerate} % Starting an enumerate environment.

    \item Easy integration of figures, tables, and mathematical equations. % Adding an item.

    \item Automatic generation of table of contents, list of figures, and list of tables. % Adding another item.

    \item Easy creation of bibliographies and citations. % Adding a final item.

\end{enumerate} % Ending the enumerate environment.

\section{Conclusion} % Starting the last section.

LATEX is a powerful and sophisticated typesetting tool that offers numerous benefits to users. % Adding the final text.

\end{document} % Ending the document environment.
