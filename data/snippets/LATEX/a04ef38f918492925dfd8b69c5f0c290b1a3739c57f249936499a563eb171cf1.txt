\section{Introduction}
LaTeX is a typesetting language used for creating professional documents such as articles, reports, and books. It is based on the plain text format, making it simple and efficient for creating complex mathematical equations and graphics.

% Begin subsection and label it for referencing
\subsection{Benefits of using LaTeX}
\label{subsec:benefits}
There are several benefits of using LaTeX for document creation, such as:

% Begin itemize to create a bullet list
\begin{itemize}
    \item High-quality typesetting: LaTeX produces documents with beautiful typography and formatting.
    \item Excellent for technical and scientific documents: Its ability to handle equations, tables, and graphics make it the preferred choice for technical and scientific documents.
    \item Easy referencing and cross-referencing: LaTeX allows for easy insertion and referencing of labels and captions, making it convenient for creating long and complex documents.
    \item Platform-independent: LaTeX can be used on any operating system, making it a versatile choice for document creation.
    \item Efficient collaboration: LaTeX files are plain text, making it easy to collaborate with others using version control systems like Git.
\end{itemize}
