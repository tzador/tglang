\documentclass{article} %creates a new document

\usepackage[utf8]{inputenc} %sets the input encoding to UTF-8

\title{My Document} %sets the title of the document

\date{\today} %sets the date of the document to the current date

\begin{document} %begins the document

\maketitle %creates a title page using the title and date set before

\section{Introduction} %creates a new section titled "Introduction"

This is my first \LaTeX{} document. %prints text and uses the \LaTeX{} command to print the LaTeX logo

\subsection{Subsection 1} %creates a subsection titled "Subsection 1"

Lorem ipsum dolor sit amet, consectetur adipiscing elit. %prints lorem ipsum text

\subsection{Subsection 2}

Etiam justo est, luctus in sem eu, maximus tempor nunc. %prints more text

\subsubsection{Subsubsection 1} %creates a subsubsection titled "Subsubsection 1"

Nulla facilisi. Aliquam erat volutpat. %prints more text

\subsubsection{Subsubsection 2}

Cras nec risus ut nisi malesuada placerat. %prints more text

\paragraph{Paragraph 1} %creates a paragraph titled "Paragraph 1"

Sed in justo ac dui suscipit congue. %prints more text

\subparagraph{Subparagraph 1} %creates a subparagraph titled "Subparagraph 1"

Donec ac lectus eget elit ultricies condimentum. %prints more text

\section{Conclusion}

In conclusion, this document showcases various formatting options available in \LaTeX{}. %prints text and uses the \LaTeX{} command to print the LaTeX logo

\end{document} %ends the document
