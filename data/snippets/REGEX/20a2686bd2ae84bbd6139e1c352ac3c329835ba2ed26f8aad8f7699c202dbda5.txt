// Match phone number in format ###-###-####
^\d{3}-\d{3}-\d{4}$ 

//Explanation: 
// ^ indicates the start of the string
// \d{3} matches 3 digits in a row
// - matches the hyphen character
// \d{3} matches 3 digits in a row
// - matches the hyphen character
// \d{4} matches 4 digits in a row
// $ indicates the end of the string
