;;Create a function that returns the sum of two numbers
(defun sum (a b)
  (+ a b))
; This is a function in Common Lisp, denoted by the 'defun' keyword
; It is named 'sum' and takes in two parameters, 'a' and 'b'
; The body of the function is enclosed in parentheses
; The '+' function is used to add the two parameters together
; The result is returned as the output of the function
; The function can be called by using the name 'sum' and passing in two arguments, as in (sum 5 10)
