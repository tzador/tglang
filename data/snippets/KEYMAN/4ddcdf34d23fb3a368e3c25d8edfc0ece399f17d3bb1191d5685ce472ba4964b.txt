+ [CTRL ALT (1, 2, 3)] any(ConsKey) + 'add' > Context

// This snippet defines a context rule in KEYMAN
// The context rule has a condition, trigger, and output
// The condition states that when the key combination CTRL + ALT + (1, 2, or 3) is pressed
// And the context of a consonant key is active
// The output will add the string 'add' to the current context
