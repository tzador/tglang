+ [CTRL K] > "I" > Insert "italic" unicode(0x1D7CB) |
"+" is the identifier for a rule in Keyman
"CTRL K" is the context for this rule, meaning it will only apply when the user presses 'CRTL' and 'K' simultaneously
">" is used to indicate a subprocess, in this case, the "I" rule
"I" is the name of the subprocess, to which we are passing an argument
"Insert" is the action keyword, indicating that we are inserting data
"italic" is the text we want to insert
unicode(0x1D7CB) is the hexadecimal code for the italic unicode character "𝟋"
