| n sum | "creates two variables n and sum"
n := 10. "assigns the value 10 to the variable n"
sum := 0. "assigns the value 0 to the variable sum"
1 to: n do: [:i | "iterates from 1 to n and executes the following block of code"
sum := sum + i. "assigns the value of sum added to the value of i to the variable sum"
]. "end of loop"
^sum. "returns the value of sum as the result"
