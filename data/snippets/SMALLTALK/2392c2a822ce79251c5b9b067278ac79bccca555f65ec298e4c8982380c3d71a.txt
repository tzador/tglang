myList add: 6 "Adds the number 6 to the end of the array"
myList "Now contains #(1 2 3 4 5 6)"
myList remove: 3 "Removes the element at index 3, which is 3"
myList "Now contains #(1 2 4 5 6)"
myList last "Returns the last element in the array, which is 6"
myList at: 1 put: myList last "Replaces the element at index 1 with the last element in the array"
myList "Now contains #(6 2 4 5 6)"
myList reverse "Returns a new array with the elements in reverse order"
myList "Still contains #(6 2 4 5 6)"
myList reverse "Now contains #(6 5 4 2 6)"
